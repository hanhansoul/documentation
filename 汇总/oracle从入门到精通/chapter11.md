# 触发器

触发器由数据库的特定事件来触发。
- INSERT
- UPDATE
- DELETE
- CREATE
- ALTER
- DROP
- 用户登录或注销
- 数据库的打开和关闭
- 特定的错误信息

触发器的作用
- 自动生成自增长的字段
- 执行更复杂的业务逻辑
- 防止无意义的数据操作
- 提供审计
- 允许或限制修改某些表
- 实现完整性规则
- 保证数据的同步复制

触发器分类
1. DML触发器
2. DDL触发器
3. 复合触发器
4. INSTEAD OF触发器，通常作用于视图
